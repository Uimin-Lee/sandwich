git command line


맥에서는 terminal 을 통해 명령어를 입력할 수 있다

git add : 투석기

git add -A : 앞으로 바뀌는 모든 사항 stage

git commit -m “설명 쓰기”

git log : 기록 조회



branch : repository 안의 repository

* git branch

* git checkout -b 이름



source tree 사용법


source tree 는 git 을 시각적으로 보기 편리하도록 도와주는 툴이다.

아주 간단하게 master 와 branch 를 생성하고 merge 할 수 있다


git hub 사용법


git 은 특이하게 코드 전체가 아니라 변경 사항만을 따로 저장하기 때문에 원격으로 여러 명이 동시에 한 코드를 수정할 때

그 장점을 발휘하는데 git hub 는 이런  git 의 장점이 더욱 발휘될 수 있도록 다른 사용자들과 코드를 공유할 수 있도록 돕는다

git hub 는 사용자에게 repository를 제공한다
